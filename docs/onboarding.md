## Step 2 — Install Node.js (one-time, skip if already installed)

Download and install from **nodejs.org** — just click the LTS version and run the installer.

Verify it worked: open Terminal and run `node --version` — you should see a version number.

## For Rehaan — Spend & Access Management

### Adding a new team member
1. **claude.ai** → Settings → Team → Invite Member (enter their work email)
2. Send them their personal setup command from Step 3 above
3. Cost: **$30/seat/month** added to your Team plan automatically

### Removing access
1. **claude.ai** → Settings → Team → remove the member
2. Their Claude access is revoked immediately — no other steps needed

### Setting a spend cap (API usage above Team plan)
1. Go to **console.anthropic.com** → Settings → Billing → Usage Limits
2. Set **Monthly Spend Limit** to `$100`
3. Set an **Email Alert** at `$80` to get notified before hitting the cap
4. This caps any overage billing — usage stops rather than charging past the limit
